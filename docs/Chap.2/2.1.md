# 2.1数据操作
操作对象：张量类

张量：一个由数值组成的数组

eg：一维张量对应向量，二维对应矩阵

## 2.1.1 入门
用pytorch控制张量

```python
import torch

x = torch.arrange(12)
#使用arange创建一个行向量x。这个行向量包含以0开始的前12个整数
x.shape
#输出：torch.Size([12]),其中torch.Size()是一个描述x形状的元组，例如torch.size([2,3])说明这个张量是一个两行三列的矩阵，而torch.size([12])说明这个张量是一个12列的行向量
x.numel()
#输出：12,表示x的元素个数，如果是一个[2,3]的矩阵，则返回6
X = x.reshape(3,4)
#reshape()函数将x重新定义为3行4列的矩阵,，但其中元素不变
若想自动计算维度，则可以写（3，-1）.
```
### 创建张量的方法
```python
torch.zeros(2,3,4)
#创建一个全零张量，形状为2行3列4列
torch.ones(2,3,4)
#创建一个全1张量，形状为2行3列4列
torch，randn(2,3,4)
#创建一个正态分布张量，形状为2行3列4列
torch.tensor([[2,1],[1,2]])
#创建一个张量，形状为2行2列，元素为2和1
torch.arange(0,10,0.1)
#创建一个从0到10的步长为0.1的向量
```

## 2.1.2 张量的运算
按元素运算：
```python
x = torch.tensor([1,2,3])
y = torch.tensor([4,5,6])
x + y #输出：tensor([5, 7, 9])
x * y #输出：tensor([4, 10, 18])
```
## 2.1.3 广播机制

1.  通过适当复制元素来扩展一个或两个数组，以便在转换之后，两个张量具有相同的形状
2. 对生成的数组执行按元素操作

eg ：
```python
a = torch.arange(3).reshape(3,1)
b = torch.arange(2).reshape(1,2)
#此时a = torch.tensor([[0],[1],[2]])转置，b = torch.tensor([[0,1]])，形状不相同。
```
此时将a复制列，b复制行，构成3*2矩阵，再逐元素相加。
```python
a + b
#输出：
tensor([[0, 1],
[1, 2],
[2, 3]])
```

## 2.1.4 索引
**注：为了和计算机索引方式匹配，所有矩阵都从第0行第0列开始索引**

由于张量的本质是多维数组，那么对于数组适用的索引等操作同样适用于张量,**但要注意索引指向的是某行的全部元素**。

如：
```python
X = torch.arange(12).reshape(3,4)
X[0] = tensor([1,2,3,4])
#矩阵第0行，索引0
X[1, 2] = 9
#矩阵第1行，第二列赋值为9
X[0:2, :] = 12
#矩阵第0行到第2行，全部列赋值为12
#[0:2, :]，前面的0:2表示从0到2的左开右闭区间，（2-0=2所以含有两个元素），后面的:表示所有列
```
