# 2.5 自动微分

**求导是几乎所有深度学习优化算法的关键步骤。**

## 反向传播
pytorch通过反向传播函数实现自动求梯度。

其适用于**输出较少**的情况。

例如：
```python
# 详细展示反向传播过程
import torch

def detailed_backpropagation():
    # 构建计算图
    x = torch.tensor(1.0, requires_grad=True)
    w = torch.tensor(2.0, requires_grad=True)
    b = torch.tensor(3.0, requires_grad=True)
    
    # 前向传播
    z = w * x + b      # z = 2*1 + 3 = 5
    loss = z ** 2      # loss = 25
    
    # 反向传播自动求导
    loss.backward()
    
    # 看看自动求导计算的梯度
    print(f"∂loss/∂x = {x.grad}")  # ∂loss/∂x = 2*z*w = 2*5*2 = 20
    print(f"∂loss/∂w = {w.grad}")  # ∂loss/∂w = 2*z*x = 2*5*1 = 10
    print(f"∂loss/∂b = {b.grad}")  # ∂loss/∂b = 2*z*1 = 10
    
    # 手动验证链式法则
    # ∂loss/∂z = 2*z = 10
    # ∂z/∂x = w = 2
    # ∂loss/∂x = ∂loss/∂z * ∂z/∂x = 10 * 2 = 20 ✓

detailed_backpropagation()
```

import torch
x = torch.tensor(1.0, requires_grad=True)
# 创建标量张量，并设置requires_grad=True，表示需要计算梯度
t = torch.tensor(3.0,requires_ grad=True)
y = x + t
y.backward()
print(x.grad)
# 输出：tensor(1.)

那么y = x + t
y.backward()
print(x.grad)
