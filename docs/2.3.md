# 2.3 线性代数

## 回顾
torch中，矩阵用二阶张量表示
```python
x = torch.randn(5, 3)
# x 是5行3列矩阵
x.T
#为3行5列矩阵
x.sum()
#为x中元素求和（降维）

```

## 小结
•标量、向量、矩阵和张量是线性代数中的基本数学对象。
• 向量泛化自标量，矩阵泛化自向量。
• 标量、向量、矩阵和张量分别具有零、一、二和任意数量的轴。
• 一个张量可以通过sum和mean沿指定的轴降低维度。
• 两个矩阵的按元素乘法被称为他们的Hadamard积。它与矩阵乘法不同。
• 在深度学习中，我们经常使用范数，如L1范数、L2范数和Frobenius范数。
• 我们可以对标量、向量、矩阵和张量执行各种操作。
