## 二、 `yield` 是什么？

`yield` 是 Python 中一个非常核心且强大的关键字，它用于**创建生成器（Generator）**。

简单来说，`yield` 可以让一个函数**"暂停"并"记住"当前状态**，下次调用时从暂停的地方继续执行，而不是从头开始。

---

### 1.1 普通函数 vs 生成器函数

| 特性 | 普通函数（`return`） | 生成器函数（`yield`） |
| :--- | :--- | :--- |
| **执行方式** | 一次性执行完毕 | 逐次执行，可暂停 |
| **返回值** | 返回一个值后函数结束 | 返回一个值后函数**暂停**，可继续 |
| **内存占用** | 一次性生成所有结果 | **惰性求值**，按需生成，节省内存 |
| **使用场景** | 计算结果 | 处理大数据流、无限序列 |

---

### 二、`yield` 的工作原理

#### 2.1 代码示例

```python
def my_generator():
    """一个简单的生成器函数"""
    print("第1步：开始执行")
    yield 1  # 暂停，返回 1

    print("第2步：继续执行")
    yield 2  # 暂停，返回 2

    print("第3步：继续执行")
    yield 3  # 暂停，返回 3

    print("第4步：全部完成")
    # 函数结束，不再返回值

# 调用生成器函数 → 返回生成器对象
gen = my_generator()
print(type(gen))  # <class 'generator'>

# 使用 next() 逐次获取值
print(next(gen))  # 输出：第1步：开始执行 → 1
print(next(gen))  # 输出：第2步：继续执行 → 2
print(next(gen))  # 输出：第3步：继续执行 → 3
print(next(gen))  # 输出：第4步：全部完成 → StopIteration 异常
```

## next() 函数
`next()` 函数用于获取生成器的下一个值。
