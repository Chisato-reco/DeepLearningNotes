# 3.1线性回归

## 3.1.1 线性回归的基本元素
线性回归基于几个简单的假设：
1. 数据是**线性关系**
2. 噪声遵循**正态分布**

在机器学习中，我们通常使用 $n$ 来表示数据集中的样本总数。对于数据集中的第 $i$ 个样本，其输入特征表示为一个二维**列向量**：

$$
\mathbf{x}^{(i)} = \begin{bmatrix} x^{(i)}_1 \\ x^{(i)}_2 \end{bmatrix}
$$

### 线性模型
线性模型是输入特征的一个**仿射变换**，特点是**通过对加权和特征进行线性变换，并通过偏置项来平移**
数学表示如下：
$$y = \mathbf{w_1}*\mathbf{x_1}+ 
\mathbf{w_2}*\mathbf{x_2} + b$$

所以对于任意数据集，目标只有一个：**寻找模型的权重$\mathbf{w}$和偏置项$b$**

如果用点积形式表达，那么为：
$$\hat{y} = \mathbf{w}^\top*\mathbf{x} + b$$
对于数据集的矩阵表示为：
$$\hat{y} = \mathbf{X}\mathbf{w} + b$$

### 损失函数
数值越小拟合效果越好
平方误差表示为：
$$l(\mathbf{w},b)^{(i)} = \frac{1}{n}\sum_{i=1}^n \frac{1}{2}\left(y^{(i)} - \hat{y}^{(i)}\right)^2$$
即样本值减预测值平方和/2

损失函数为了度量整个模型的质量，通常使用**均值**：
$$
L(\mathbf{w}, b) = \frac{1}{n} \sum_{i=1}^n \ell^{(i)}(\mathbf{w}, b) = \frac{1}{n} \sum_{i=1}^{n} \frac{1}{2} \left( \mathbf{w}^\top \mathbf{x}^{(i)} + b - y^{(i)} \right)^2.
$$
我们希望通过训练找到一组最优参数，使得模型在整个数据集上的损失最小：

\[
\mathbf{w}^*, b^* = \arg\min_{\mathbf{w}, b} L(\mathbf{w}, b)
\]

### 随机梯度下降
在无法获取解析解时，，可以使用随机梯度下降，几乎适用于所有深度学习模型的优化。
原理：梯度是函数变化最快的方向，反过来减去梯度方向上的向量就可以让函数单调递减。

## 3.1.2 矢量化加速

矢量化的好处：**利用线性代数库取代开销高昂的for循环**

在深度学习的模型训练中，应始终使用向量化运算处理整个小批量的样本，而非使用Python循环逐样本处理——这是代码能获得数量级性能提升的关键

## 3.1.3 正态分布和平方损失

似然：已经抛了10次，出现7次正面（数据已知），问这枚硬币正面朝上的概率是多少？

也就是**已知观测数据推断参数的可能值**

正态分布的公式：
$$f(x) = \frac{1}{\sqrt{2\pi\sigma^2}}e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$
改变均值会产生沿x轴的偏移，增加方差将会分散分布、降低其峰值。

在前面的线性回归中，我们使用平方误差作为损失函数，这样的均方形式可用于线性回归的原因是**我们假设其中包含了噪声，且噪声遵循正态分布**。

推导可知，**最小化均方误差等价于对线性模型的极大似然估计**。
