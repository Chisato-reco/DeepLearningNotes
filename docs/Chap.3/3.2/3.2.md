# 3.2 线性回归的从零开始实现

只使用张量和自动求导实现。

## 3.2.1 生成数据集

生成数据集包含1000个样本，每个样本包含两个特征，均从标准正态分布中采样。

得到模型：
$$
\mathbf{y} = \mathbf{X} \mathbf{w} + b + \boldsymbol{\epsilon}
$$
其中 $\boldsymbol{\epsilon}$ 是噪声，认为它服从均值0的正态分布。

``` python
def synthetic_data(w, b, num_examples): #@save
"""生成y=Xw+b+噪声"""
X = torch.normal(0, 1, (num_examples, len(w)))
#生成符合正态分布的x矩阵，shape为(num_examples, len(w))
y = torch.matmul(X, w) + b
#做矩阵乘法并给出线性拟合式
y += torch.normal(0, 0.01, y.shape)
#在预测值基础上加上噪声
return X, y.reshape((-1, 1))
#把y变成列向量输出便于计算且统一形式
#--------定义线性回归计算函数----------#
true_w = torch.tensor([2,-3.4])
true_b = 4.2
features, labels = synthetic_data(true_w, true_b, 1000)
print('features:', features[0], '\nlabel:', labels[0])
```
输出：
``` python
features:tensor([1.4632,0.5511])
label:tensor([5.2498])
```

## 3.2.2 读取数据集

```python
def data_iter(batch_size, features, labels):
    """
    小批量随机梯度下降数据迭代器
    
    参数:
        batch_size (int): 每个小批量的样本数量
        features (torch.Tensor): 特征矩阵，形状为 (num_examples, num_features)
        labels (torch.Tensor): 标签向量，形状为 (num_examples, 1)
    
    返回:
        yield (X_batch, y_batch): 每次迭代产出一个小批量的特征和标签
    """
    # 1. 获取样本总数
    num_examples = len(features)  # features 的第一维度大小（行数）
    
    # 2. 生成索引列表 [0, 1, 2, ..., num_examples-1]
    indices = list(range(num_examples))
    
    # 3. 随机打乱索引顺序（实现随机采样，避免模型学习到数据顺序）
    random.shuffle(indices)
    
    # 4. 按批次大小遍历所有样本
    for i in range(0, num_examples, batch_size):
        # 4a. 取出当前批次的索引（注意处理最后一个不完整的批次）
        batch_indices = torch.tensor(
            indices[i: min(i + batch_size, num_examples)]
            #indices是列表，indices[]是对其进行切片，例如indices[3,7]
        )
        
        # 4b. 根据索引从 features 和 labels 中取出对应的数据
        yield features[batch_indices], labels[batch_indices]
```
## 3.2.3 初始化模型参数
模型参数的初始化，这里使用均值为0，标准差为0.01的正态分布初始化。

```python
w = torch.normal(0, 0.01, size=(2,1), requires_grad=True)
b = torch.zeros(1, requires_grad=True
)
```
初始化之后，在之后只需更新参数即可。**每次更新都要计算损失函数关于模型参数的梯度。**

## 3.2.4 定义模型
```python  
def linreg(X, w, b): #@save
"""线性回归模型"""
return torch.matmul(X, w) + b
```

## 3.2.5 定义损失函数
由于要计算损失函数的梯度，所以在此定义损失函数。
```python
def squared_loss(y_hat, y): #@save
"""均方损失"""
return (y_hat- y.reshape(y_hat.shape)) ** 2 / 2
```

## 3.2.6 定义优化算法

``` python
def sgd(params, lr, batch_size):  #@save
    """小批量随机梯度下降"""
    with torch.no_grad():
        # 禁用梯度跟踪，在此上下文中的所有操作都不会被记录到计算图中。防止内存爆炸。
        for param in params:
            param -= lr * param.
            grad / batch_size
            ## 核心计算公式
            param.grad.zero_()
            ##清空梯度值
```
**为什么需要 `with torch.no_grad()`禁用梯度跟踪？**
+ 参数更新操作（param -= ...）不需要计算梯度
+ 如果不禁用，PyTorch 会尝试追踪这些操作，导致内存泄漏和计算图爆炸
+ 这是 PyTorch 中更新参数的标准写法

## 3.2.7 训练
在每次迭代中，我们读取一小批量训练样本，并通过我们的模型来获得一组预测。计算完损失后，我们开始反向传播，存储每个参数的梯度。最后，我们调用优化算法sgd来更新模型参数。

每次训练循环如下：
+ 初始化参数
+ 重复以下训练：
    - 计算梯度
    $$
        \mathbf{g} \leftarrow \partial_{(\mathbf{w}, b)} \frac{1}{|\mathcal{B}|} \sum_{i \in \mathcal{B}} \ell\left(\mathbf{x}^{(i)}, y^{(i)}, \mathbf{w}, b\right)
        $$
    - 更新参数
         $$
            (\mathbf{w}, b) \leftarrow (\mathbf{w}, b) - \eta \, \mathbf{g}
            $$

在每个**迭代周期**（epoch）中，我们使用`data_iter`函数遍历整个数据集，并将训练数据集中所有样本都使用一次。

其中超参数lr学习率，num_epochs迭代周期数。

```python
lr = 0.03
num_epochs = 3
net = linreg
#把模型封装为net
loss = squared_loss
#把损失函数封装为loss
```

```python
for epoch in range(num_epochs):
for X, y in data_iter(batch_size, features, labels):
l = loss(net(X, w, b), y) # X和y的小批量损失
# 因为l形状是(batch_size,1)，而不是一个标量。l中的所有元素被加到一起，
# 并以此计算关于[w,b]的梯度
l.sum().backward()
sgd([w, b], lr, batch_size) # 使用参数的梯度更新参数
with torch.no_grad():
train_l = loss(net(features, w, b), labels)
#每循环一次计算一次损失
print(f'epoch {epoch + 1}, loss {float(train_l.mean()):f}')
```

输出：
```

epoch 2, loss 0.000162
epoch 3, loss 0.000051
```

## 小结
线性回归的基本步骤：
+ 获取并读取数据集
+ 初始化模型参数
+ 定义模型（线性函数模型）
+ 定义损失函数（平方均值）
+ 定义优化算法（梯度下降）
+ 训练模型（定义训练循环，反复训练减小loss）
